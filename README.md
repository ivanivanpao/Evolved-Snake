## 🎨 設計理念 (Design Concept)
本專案將經典的「貪食蛇」重新想像為一場抗陽防曬、採集棉花賺錢去美白的趣味故事冒險！我們希望打破傳統貪食蛇單調的吃點得分機制，引進了**「隨著分數增長，陽光越來越耀眼，最終將玩家曬黑」**的故事張力。

遊戲融合了現代網頁的極致美學（Rich Aesthetics）與幽默搞笑的沉浸式互動：從剛開局需要「戴上墨鏡 😎 避光」的遮罩儀式，到 300 分後世界變為極黑棉花夜、食物化為白色發光棉花 `☁️`。我們運用了 Web Audio API 純代碼實時合成了復古紅白機清脆音效，更將太陽特效的渲染時長加長至具有電影感張力的 3.5 秒。透過這些細節，我們旨在打造一款視覺華麗、操作爽快且充滿驚喜的現代化休閒網頁遊戲！

## 🚀 核心差異：我的版本與原版有何不同？
1. **沉浸式故事劇本 (300分週期無限迴圈)**：
   不再是無止盡的吃食物！玩家將經歷：**陽光漸強 ➔ 150分日食異象 ➔ 300分全身曬黑（食物變棉花、暫停畫面黑洞化） ➔ 350/400分努力收集棉花離美白更近 ➔ 450分美白成功 ➔ 重新挑戰** 的驚奇冒險！
2. **搞笑與沉浸式互動**：
   - 開局強制警告：**「為避免陽光照射，請戴上墨鏡遊玩」** 😎。這不僅緊扣主題，更完美繞過了瀏覽器的 Autoplay 音效限制！
   - 300 分後暫停畫面黑化，顯示爆笑台詞：**「你曬黑了，請收集棉花賺錢，去做美白」**！
3. **極致聲光特效與無延遲音效**：
   - 全螢幕太陽慶祝特效：隨機播放自訂的中國風魔性音效，伴隨慢速旋轉的太陽光暈。
   - Web Audio API 吃食物音效：由瀏覽器晶片實時合成 C5 到 G5 的 C-bit 滑音，零延遲，悅耳無比！
4. **現代化科技感控制面板**：
   - 自主切換格子線 `⊞`
   - 加速功能開關：⚡（開啟）或 🛡️（安全防禦盾，關閉加速）
   - 🔊/🔇 音效自主開關
5. **撞牆不死機制**：
   - 蛇頭撞牆時不會結束遊戲，而是會從對側**「穿透出現」**！只有撞到自己的身體才會 Game Over 💀，大幅提升遊戲的順暢度與可玩性。

## 🎮 操作說明 (How To Play)
* **移動控制**：使用鍵盤 `W`, `A`, `S`, `D` 或 **方向鍵** 控制蛇的移動方向。
* **暫停 / 繼續**：按下 `空白鍵 (Space)` 隨時切換暫停狀態。
* **開始 / 重新開始**：撞擊身體結束後點擊「重新開始」，或在等待畫面按任意方向鍵直接出發！

## 🏆 核心遊戲規則與關卡說明

| 分數 (Score) | 關卡階段與太陽異象 | 遊戲畫面與玩法變化 |
| :--- | :--- | :--- |
| **0 ~ 140 分** | **暖陽初現** | 每 50 分觸發「普通太陽」特效，大字顯示「太陽越來越耀眼」。 |
| **150 ~ 290 分** | **日食初現** | 太陽中央微微重疊黑色太陽殘影。每 50 分文字依序顯示：「好像出現了什麼?」 ➔ 「越來越明顯」 ➔ 「快看清楚了」。 |
| **300 分** | **天黑黑！你被曬黑了 😱** | 觸發「純黑太陽」全螢幕特效。食物轉變為蓬鬆發光的**「白色棉花 ☁️」**，暫停畫面全面黑化，要求玩家收集棉花賺錢去美白！ |
| **350 & 400 分** | **黑白太陽重疊（離美白更近）** | 在棉花黑夜中，全螢幕特效呈現**「聖潔白太陽 + 黑色太陽抖動殘影」**的雙重奇觀，大字顯示「離美白越來越近」。 |
| **450 分** | **美白成功！✨** | 觸發「美白成功」大特效，渲染極致多層發光的**聖潔銀白太陽**！特效結束後世界重置，食物重回金黃太陽，開始下一輪挑戰！ |
| **500 ~ 590 分** | **太陽再次耀眼** | 經歷美白成功後，太陽重新普照大地。 |
| **600 分以上** | **故事無限輪迴** | 重回 300 分週期的曬黑 ➔ 採棉花 ➔ 450 分美白無限故事循環！ |
