## Design Concept
一般的貪吃蛇就是吃了會變長，然後撞到自己或牆壁就會死亡，實在有點無趣
所以我嘗試加入一些新元素跟設計新的遊戲玩法，同時在遊戲中加入一些有趣的元素跟音效，玩一些梗
一開始只有經典模式，後來覺得這樣不太夠，就加入了Boss模式，再加入最近學的離散數學元素
希望遊戲可以不要只有單純的移動，可以再多一點挑戰跟樂趣，同時用很炸裂的元素給玩家帶來更有趣的體驗

## How To Play
簡單的通過wasd跟上下左右去操作方向
遊戲上有兩個模式，一個經典模式，一個是Boss模式
兩個模式都有三個技能，透過太陽能量去使用
對應按鍵及效果如下
shift: 太陽神速(加速)
q: 烈日緩速(減速，針對砲彈)
e: 日冕護盾(無敵/白金流光/穿透)

Boss模式會有一個Boss，需要打敗它才能獲勝

另外太陽能量在經典可以通過吃太陽跟棉花得到分數來獲取，在Boss模式可以透過回答數學問題跟吃棉花有機率來獲取
也有一些設定可以調整音效跟背景音樂等
